Data Schema

The exported data follows a strict schema found in the global QuestDataExport table.

QuestDataExport.quests["Quest Name"] = {
    id = 12345,
    name = "Quest Name",
    level = 25,
    questText = "Full quest description...",
    objectivesText = "Slay 10 Rats...",
    giverName = "Guard Thomas",
    giverLocation = { x = 45.2, y = 33.1, zone = "Elwynn Forest", map = 1 },
    rewards = {
        money = 500, -- Copper
        xp = 250,
        items = {
            { id = 123, name = "Shiny Sword", count = 1, quality = 2 }
        }
    }
}

How to Use in Your Addon

To use the specific data collected by QDE in your own addon, follow these steps:

1. Add to .toc Dependency

## OptionalDeps: QuestDataExporter

2. Access Data Safely

local function GetQuestInfo(questName)
    if not QuestDataExport or not QuestDataExport.quests then return end
    
    local data = QuestDataExport.quests[questName]
    if data then
        print("Quest Level:", data.level)
        print("Starts at:", data.giverName)
    end
end
